# AI_POWERED_PERSONAL_BOOKING_AGENT_USING_LANG_GRAPH

## 🚀 Domain
LLM Agents | LangGraph | Scheduling Assistant | Tool Calling | Conversational AI

---

## 📌 Problem Statement

Design and develop an AI-powered personal booking and scheduling agent that can interact with users in natural language to manage meetings, appointments, and bookings.

The agent should:
- Understand user intent
- Maintain conversational context
- Resolve scheduling conflicts
- Confirm bookings using a structured, agentic workflow

---

## 🎯 Key Objectives

- Understand scheduling intents from natural language
- Maintain conversation state and user preferences
- Check availability using calendar data (real or mock)
- Handle conflicts and suggest alternative time slots
- Confirm and summarize bookings
